Lumia 435 Open Box

Sells for 80AUD, brand new.
Ok, let’s see the specifications.

CPU：Qualcomm Snapdragon 200

（Four core processor, single speed can up to 1.2 GHz）（ 4x ARM® Cortex™ A7）

GPU：Adreno 302

Screen：A 480*800 WVGA TFT Screen, 235dpi

RAM：1GB LPDDR2

ROM：8GB

Battery：1560mAh

Camera：rear 2 million，front 0.3 million……

Can use SD card，already have a 32GB Samsung SD orange card.

System：Windows 10 Mobile 16055

Let’s talk about experience first.

At first is Windows phone 8.1, I am a capital of the secretly. Because I never use Windows Phone.

After trun on, it is the same with Android, it needs input many data, but it sucks at the installing apps…My mood is really a collapse, a cell phone to take back actually have to wait 30 minutes to use … … (here refers to the Windows Phone 8.1)

I am not interested to experience WP 8.1, direct upgrade Win10M.

Then I was stuck in the upgrade assistant. Yes, my account is under 18 years old, Microsoft, you …

So I go to apply for a new Microsoft account …

And then upgrade with a full two or three hours … …

Then I was familiar with the Windows 10 interface.

Of course not finished, just upgrade the version number is 10586.1xx (do not remember), have to upgrade 14393 … … another several hours … …

—————Above is that you just got the new Lumia should do.—————

Then, what I surprise is that this phone won’t heat

Really, it won’t heat.

It seems funny. How can Qualcomm Snapdragon 200 be heat? After all the performance is so weak.

Win10M looks very similar with Windows 10, so some of the computer can not explain why the design of the place on the phone is extremely clear, such as the background of the design, there is a waste of space magnetic paste … …

But not polite to say, WP 8.1 design and layout is really sucks, all comes with large fonts, until the Win10M design came out is better, I’m not using Nokia 3310 isn’t it?

In general, the application of less … … features less, but it shows what a UWP phone should have, and UWP unlimited possibilities.

For sure…

Only one battery, comes with the charging head is not even a USB charging head plus a data cable, but even up … …

Charging speed is not a big deal, do not say something like 5V2A, and comes with the charging head actually only have poor 5V0.5A around … …

I can only say that Microsoft does have the strength to use snapdragon 410 to make the same experience with snapdragon 600 on Android.

Of course, UWP is also imperfect, such as Microsoft will never tell you what happened, will only give you a wrong code, such as UWP does not support the input method.

Cortana in the search inside the start, the ace application to start hidden so deep, I am curious Microsoft is silly or not … …

Battery life can totally support for whole day, consider it only have 1560mAh.

Sometimes will have “recoverying”.

There is no problem with Asphalt 8. Runs stable. Not joking. I only can say that barely able to open, but the number of frames is too bad.

I really do not know the boast Qualcomm’s Adreno 302 or the boast UWP optimization …

APP is still a problem. Too less.

Overall as a low-end machine, it played beyond the level I expected.

This is what I found a small bright spot: double-click can actually bright screen! This is the ultra-low-end mobile phone … … Microsoft what are you thinking … …

But there is a crash thing: the status bar no remaining battery percentage! Be sure to drop down to see!

Status bar battery percentage is more important than the double-click wake up!
