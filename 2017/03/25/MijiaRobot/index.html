Xiaomi MIJIA robot vacuum cleaner

Posted on 2017-03-25 Edited on 2017-05-10 In MiRobot Disqus:
Xiaomi Mi robot vacuum cleaner

Mi robot has been half month in my house, I think I can talk about the experience of the robot now.

View

Simple appearance, simple design, nice looking.

Get started

All Mijia series of products are basically through the Mijia APP to manage, praise.

Mijia APP function fairly rich, praise.

Only download when you need it, praise.

Mi’s server give it negative feedback, overseas users have to refresh again and again in order to download the extension, negative feedback. (By the way, Xiaomi’s official website mi.com is so bad, because Xiaomi’s cache server domain name is .cn, too slow.)

Real test

Have to say that the Mijia APP itself is easy to use and SLAM algorithm is really not a joke.

This is a map after the complete sweep.

First of all we see: the upper left corner of the part because there is a sliding door restrictions, that part can be ignored.

The middle of the two semicircle there are some inexplicable overflow, I can only say that this is not robot’s problem, because the two semi-circular walls are not ordinary walls, the two parts are glass … … so it is estimated that the machine can not scan there…

In general, the robot sweep it very full, very fine. Every corner can guarantee cleaning.

Battery Life and speed

In one minute it can sweep 0.95 square meters, run out of 19% of the electricity left, and sure enough the robot’s battery is really powerful.

Price

My dad purchase this at first.

7 in 1 Vacuum Cleaner LED Robotic Automatic Recharge Floor Sweeper Mop Robot Red | eBay

This rubbish still sells in 349.95AUD.

After we buy it we find that it always stuck.

And the Mijia Robot only cost 1699 RMB (appx. 333AUD)

Firmware update

There is a firmware 3.3.6_003028, in this firmware they add a function that can remote the robot.

Maintenance issues

Recently it can only around a small circle around their own sweep a small circle to go back, have a check after consulted the engineer, clean up the corner of the dust just fine.

Only can say that it needs good care and maintenance. _(:зゝ∠)_

In summary, the price of 1699, I think, is quite worth buying.

(When the robot stuck by the floor, you can use the APP to force restart it.)
